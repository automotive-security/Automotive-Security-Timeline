"汽车信息安全事件时间轴" 按照时间发生的先后顺序记录汽车信息安全事件。
- 地址: [Automotive-Security-Timeline](https://autosec-timeline.delikely.eu.org) 

事件记录的格式如下：
```json
{
	"media": {
	"caption": "<a href=\"原文链接\">原文标题</a>",
	"credit": "",
	"url": "图片或视频地址"
	},
	"text": {
		"headline": "事件标题",
		"text": "事件描述"
	},
	"start_date": {
		"month": "月",
		"day": "日",
		"year": "年"
	},
	"group":"vulnerability 或者 event "
}
```
