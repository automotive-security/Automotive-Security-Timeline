## Automotive-Security-Timeline
汽车信息安全事件时间轴 [timeline.icvsec.com](http://timeline.icvsec.com)

事件记录的格式如下：
```json
{
	"media": {
	"caption": "<a href=\"原文链接\">原文标题</a>",
	"credit": "",
	"url": "图片或视频地址"
	},
	"text": {
		"headline": "事件标题",
		"text": "事件描述"
	},
	"start_date": {
		"month": "月",
		"day": "日",
		"year": "年"
	},
	"group":"vulnerability 或者 event "
}
```
